## 项目简介

维ICP备案系统是一款💖开源、🩵界面美观的第三方icp虚拟备案系统

##环境支持
PHP >= 7.4
MySQL >= 5.6
Nginx 或 Apache
Linux 或 Windows 服务器

## 系统功能

### 首页功能
- 备案查询：按域名/备案号查询备案信息
- 信息公示：查看已通过备案的网站列表
- 关于页面：常见问题、备案政策说明

### 管理端功能
- 备案审核：审核备案申请，通过或驳回
- 备案申请：管理员可以代为提交备案申请
- 数据统计：备案通过率、申请趋势统计
- 系统设置：备案号生成规则、敏感词过滤
